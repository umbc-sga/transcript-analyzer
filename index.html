How to Use This App

This app was created to allow people to search and sort data from their transcripts to check graduation requirements, as well as to view trends in the data.

If you click on the table headers, it allows you to sort the table by the data in that column. To search for specific data, use the filter dropdown and input elements. The search value supports OR statements by using the pipe character ("|"). See the example queries below on how to use it.

When you apply a filter, it shows a subset of your data. If you have applied a filter, the next filter will be applied on that subset that is showing in the table, thus refining your data even more. This is called "filter stacking". This is why the search value supports OR statements, but not AND statements, because you can just stack filters. Pressing the Reset Filter button will go back to showing all of your transcript data.

Example Useful Queries:

  • Select items where Course Number is greater than or equal to to 300.
    • This query will select items where the Course Number is greater or equal to 300, in other words, it will show all of your 300-400 level courses.
  • Select items where Department equals PHIL.
    • This query will select items where the Department equals PHIL, in other words, it will show all of your classes that you've taken in the PHIL department.
  • Select items where Grade does not equal -.
    • This query will select items where Grade does not equal -, in other words, it will show all of your classes where you have a grade for it.
  • Select items where Grade equals A | B | C.
    • This query will select items where Grade equals A or B or C, in other words, it will show all of your classes where you've gotten an A, B, or C in it.
  • Select items where Semester includes Spring.
    • This query will select items where Semester includes Spring, in other words, it will show all of your classes that you've taken in a Spring semester, ignoring the year.
